# 2. 儿科学的基础和临床特点

## 2.1. Abstract -- 儿科学的特点
- 个体, 性别, 年龄之间差异大
- 儿童处于不断发育成长的阶段，疾病中的恢复速度快
- 儿童的临床治疗的重点在于预防，要处处从保健预防出发

## 2.2. 基础医学方面

1. 体格结构、解剖方面
2. 生理生化方面
3. 防病抗病免疫方面
4. 疾病的发生、发展和病理反应方面
5. 神经——心理行为方面

## 2.3. 临床医学方面

1. 疾病种类
2. 临床表现
3. 诊断
4. 治疗
5. 预后
6. 预防
